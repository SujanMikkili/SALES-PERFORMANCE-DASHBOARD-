# 📊 Sales Performance Dashboard
> **Power BI · Excel · SQL** — End-to-end Business Intelligence Dashboard

---

## 📌 Project Overview

An end-to-end **Sales Performance Dashboard** built using **Power BI Desktop**, with **Excel** as the raw data source and **SQL** for data validation, aggregation checks, and KPI verification.

The dashboard provides actionable insights into sales trends, profitability, category performance, and regional analysis — closely reflecting real-world business reporting requirements.

---

## 🎯 Objectives

- Analyze overall sales and profit performance
- Track **Month-over-Month (MoM)** sales growth
- Identify top and bottom performing categories and sub-categories
- Evaluate regional and customer segment performance
- Enable interactive filtering to support business decision-making

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|---|---|
| **Power BI Desktop** | Data modeling, DAX, dashboard development |
| **Excel** | Raw sales dataset |
| **SQL (SQL Server / MySQL)** | Data validation, aggregations, KPI verification |
| **Power Query** | Data cleaning and transformation |
| **DAX** | KPI and time-intelligence calculations |

---

## 📋 Dashboard Pages

### 1️⃣ Sales Overview
- Total Sales, Total Profit, Profit Margin (%)
- Month-over-Month Sales Growth
- Monthly sales trend analysis

### 2️⃣ Sales by Category & Sub-Category
- Sales vs Profit analysis by category
- Top 5 and Bottom 5 sub-categories by sales
- Profit margin analysis to identify loss-making products
- Filters by year and region

### 3️⃣ Sales by Region & Segment
- Segment-wise sales distribution
- Regional sales ranking
- Regional profit contribution analysis

---

## 📐 Key Metrics & DAX Calculations

| Metric | Description |
|---|---|
| **Total Sales** | Sum of all sales transactions |
| **Total Profit** | Sum of profit across all orders |
| **Profit Margin (%)** | Profit / Sales × 100 |
| **Previous Month Sales** | Time-intelligence calculation |
| **MoM Sales Growth (%)** | (Current Month - Previous Month) / Previous Month |

> ✅ All key metrics were validated using SQL queries to ensure data accuracy.

## 🚀 How to Use

1. Download the Power BI file from the `/PowerBI` folder
2. Open it using **Power BI Desktop**
3. Use slicers and visuals to explore insights
4. Review SQL queries in the `/SQL` folder for data validation logic

📊 Sales Performance Dashboard
Power BI | Excel | SQL (Data Validation & Analysis)

📌 Project Overview
This project is an end-to-end Sales Performance Dashboard developed using Power BI Desktop, with Excel as the raw data source.
SQL was used for data validation, aggregation checks, and KPI verification to ensure accuracy and consistency of analytical results.

The dashboard provides insights into sales trends, profitability, category performance, and regional analysis, closely reflecting real-world business reporting requirements.

🎯 Objectives
Analyze overall sales and profit performance
Track month-over-month (MoM) sales growth
Identify top and bottom performing categories and sub-categories
Evaluate regional and customer segment performance
Enable interactive filtering to support business decision-making
🛠️ Tools & Technologies
Power BI Desktop – Data modeling, DAX, dashboard development
Excel – Raw sales dataset
SQL (SQL Server / MySQL) – Data validation, aggregations, KPI verification
Power Query – Data cleaning and transformation
DAX – KPI and time-intelligence calculations
